# 🎮 Scratch Community Platform

> 面向 K12 的一站式 Scratch 编程社区平台 — 融合「创作 + 分享 + 刷题 + 竞赛 + 积分 + AI」

## 📌 项目简介

本项目旨在打造一个开源的少儿编程社区平台，区别于市面上功能割裂的竞品，将以下能力整合为一体：

- 🎨 **在线创作** — Scratch 3.0 / Python / Blockly 在线编辑器
- 🌐 **社区分享** — 作品发布、点赞、收藏、评论、Remix、关注
- 📝 **OJ 刷题** — Scratch/Python/C++ 自动判题，支持 SPJ 特判
- 🏆 **竞赛系统** — 定时赛、创意赛、组队赛、段位赛 1v1
- ⭐ **积分成就** — 积分/等级/徽章/排行榜 完整激励体系
- 🤖 **AI 原生** — AI 编程助手、作品点评、自动出题、学习路径推荐
