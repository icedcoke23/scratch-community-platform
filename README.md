# 🎮 Scratch Community Platform

> 面向 K12 课后教学的 Scratch 编程社区 — 融合「创作 + 社区 + 刷题 + 作业 + 积分 + AI」

## 📌 项目简介

开源的 Scratch 编程社区平台，面向课后教学场景：

- 🎨 **在线创作** — Scratch 3.0 编辑器（基于 scratch-vm），云变量同步
- 🌐 **社区分享** — 作品发布、点赞、收藏、评论、关注、排行榜
- 📝 **OJ 刷题** — Scratch 自动判题（sb3 解析 + scratch-vm 运行 + AI 评分）
- 📚 **课后教学** — 作业布置/提交/批改、测评、学情报告
- ⭐ **班级宠物** — 趣味化积分激励，全班共同养育宠物
- 🤖 **AI 原生** — AI 点评、AI 判题、AI 出题、AI 学情分析

## 🏗️ 架构

```
部署单元 = 3 个:
  1. 模块化单体 (Spring Boot 3) — 12 个业务模块 + sb3 解析库 + 判题核心库
  2. 判题沙箱 (Node.js + Docker) — scratch-vm 隔离执行
  3. AI Gateway (FastAPI) — LLM 调度/缓存/降级
```
