# 🎮 Scratch Community Platform

> 面向 K12 的 Scratch 编程社区平台 — 融合「创作 + 社区 + 刷题 + 作业 + 积分 + AI」

## 📌 项目简介

本项目旨在打造一个开源的 Scratch 编程社区平台，面向课后教学场景，核心能力：

- 🎨 **在线创作** — Scratch 3.0 在线编辑器，基于 scratch-vm
- 🌐 **社区分享** — 作品发布、点赞、收藏、评论、关注
- 📝 **OJ 刷题** — Scratch 自动判题（sb3 解析 + AI 分析）
- 📚 **课后教学** — 作业布置/提交/批改、测评、班级管理
- ⭐ **积分体系** — 班级宠物积分、多维度数据统计
- 🤖 **AI 原生** — AI 编程助手、作品点评、自动出题、学情分析

> 当前阶段聚焦 Scratch，后续可扩展 Python、C++ 等语言。
