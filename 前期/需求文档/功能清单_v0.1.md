# 📝 功能清单 v0.1

> 日期：2026-04-23 | 聚焦 Scratch + 课后教学

---

## 一、核心功能模块

### 1. 用户系统
- [ ] 注册/登录（邮箱/手机）
- [ ] 个人主页（作品/成就/统计数据）
- [ ] 关注/粉丝体系
- [ ] 角色管理（学生/教师/管理员）

### 2. Scratch 创作引擎
- [ ] Scratch 3.0 在线编辑器（基于 scratch-vm）
- [ ] 素材库（角色/背景/声音）
- [ ] 云变量实时同步
- [ ] 自动保存
- [ ] 项目导出 (.sb3)

### 3. 社区系统
- [ ] 作品发布（标题/描述/标签/封面/可见性）
- [ ] Feed 流（关注/推荐/热门）
- [ ] 点赞/收藏/评论
- [ ] 标签体系（游戏/动画/故事/艺术/音乐）
- [ ] 排行榜（周/月/总）

> ❌ 不做工作室/组队创作

### 4. ⭐ sb3 解析引擎
- [ ] 自动解压 .sb3 文件
- [ ] 解析 project.json
- [ ] 提取角色信息（名称/坐标/大小/方向/造型/声音）
- [ ] 提取舞台信息（背景/全局变量/列表/广播）
- [ ] 提取程序链（积木序列/触发方式/嵌套结构）
- [ ] 提取素材清单（所有 png/svg/wav/mp3）
- [ ] 提取变量/列表/广播及关系
- [ ] 积木类型统计
- [ ] 复杂度评分
- [ ] 结构化 JSON 输出 → MongoDB
- [ ] 可视化展示解析结果

### 5. OJ 刷题
- [ ] 题目管理后台
- [ ] Scratch 算法题（sb3 解析 → 注入测试用例 → VM 运行 → 比对）
- [ ] Scratch 创意题（sb3 解析 → AI 评分）
- [ ] 选择题/填空题
- [ ] 测试数据图形化生成器
- [ ] 部分分（按测试点）
- [ ] 题解 + 讨论区
- [ ] 错题本

### 6. 课后教学
- [ ] 作业布置（教师指定班级/截止时间）
- [ ] 作业提交（学生提交 sb3 → 自动解析）
- [ ] 自动批改（关联题目时自动评分 + AI 点评）
- [ ] 手动批改（教师查看 sb3 解析结果 → 打分 + 评语）
- [ ] 测评（定时测验，自动评分）
- [ ] 截止提醒
- [ ] 学情报告（学生个人 + 班级整体）

### 7. 班级宠物积分
- [ ] 积分获取（登录/创作/刷题/作业/社交）
- [ ] 个人等级体系（6 级）
- [ ] 班级宠物（全班共同喂养，宠物成长进化）
- [ ] 宠物等级体系（6 级）
- [ ] 徽章系统

### 8. 多维度学情统计
- [ ] 编程能力（积木种类/复杂度/嵌套深度/变量使用）
- [ ] 学习进度（课程完成率/作业提交率/正确率趋势）
- [ ] 创作能力（作品数量/素材多样性/原创度）
- [ ] 刷题数据（AC率/知识点掌握/错题分布）
- [ ] 班级对比（排名/班级间对比/宠物等级）
- [ ] AI 洞察（薄弱点/学习建议/进步预测）

### 9. AI 能力
- [ ] AI 作品点评（接收 sb3 解析数据 → 生成点评）
- [ ] AI 自动判题（创意题评分）
- [ ] AI 编程助手（对话式辅助）
- [ ] AI 出题助手
- [ ] AI 学情分析
- [ ] AI 内容审核

### 10. 管理后台
- [ ] 用户管理
- [ ] 班级管理
- [ ] 内容审核
- [ ] 数据统计

## 二、MVP 路线图

### Phase 1 — 基础社区 (4-6 周)
- 用户注册/登录
- Scratch 在线编辑器
- 作品发布/展示/点赞/收藏/评论
- 个人主页/关注
- 基础排行榜

### Phase 2 — sb3 解析 + OJ (4-6 周)
- sb3 解析引擎（核心模块）
- 题目管理后台
- Scratch 算法题判题
- 创意题 AI 评分
- 错题本

### Phase 3 — 课后教学 (3-4 周)
- 作业布置/提交/批改
- 测评系统
- 班级管理
- 学情报告

### Phase 4 — 积分 + AI (4-6 周)
- 积分系统
- 班级宠物
- 等级/徽章
- AI 作品点评
- AI 编程助手
- 多维度学情统计

### Phase 5 — 高级功能 (持续迭代)
- AI 出题/学情分析
- 移动端适配
- 后续扩展 Python/C++
