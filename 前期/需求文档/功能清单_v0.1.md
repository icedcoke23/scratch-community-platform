## 二、MVP 路线图

### Phase 1 — 基础社区 (4-6 周)
- 用户注册/登录
- Scratch 在线编辑器
- 作品发布/展示/点赞/收藏/评论
- 个人主页/关注体系
- 基础排行榜

### Phase 2 — OJ + 刷题 (4-6 周)
- 题目管理后台
- Scratch 算法题判题引擎
- Python 题判题
- 题单/分类/难度筛选
- 错题本

### Phase 3 — 竞赛 + 积分 (3-4 周)
- 定时竞赛
- 实时排名
- 积分系统
- 等级/徽章

### Phase 4 — AI + 课程 (4-6 周)
- AI 作品点评
- AI 编程助手
- 课程体系
- 学习地图

### Phase 5 — 高级功能 (持续迭代)
- Remix 机制
- 协作编辑
- 段位赛 1v1
- 移动端/小程序
- 硬件扩展积木
- 插件市场
