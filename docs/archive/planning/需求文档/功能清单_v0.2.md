# 📝 功能清单 v0.2

> 日期：2026-04-23 | 对齐 v0.2 架构

---

## 一、功能模块（10 个）

### 1. 用户系统
注册/登录、个人主页、关注/粉丝、角色（学生/教师/管理员）

### 2. Scratch 创作引擎
Scratch 3.0 编辑器（scratch-vm 魔改）、素材库、云变量（WebSocket）、自动保存、导出 .sb3

### 3. 社区系统
作品发布（标题/描述/标签/封面/可见性）、Feed 流（关注/推荐/热门）、点赞/收藏/评论、排行榜（周/月/总）

### 4. ⭐ sb3 解析引擎（共享库）
自动解压 .sb3 → 解析 project.json → 提取角色/舞台/程序链/变量/列表/广播/素材 → 复杂度评分 → 结构化 JSON

### 5. OJ 刷题
Scratch 算法题（sb3 解析 → 判题沙箱运行 → 比对输出）、Scratch 创意题（sb3 解析 → AI 评分）、选择/填空题、错题本

### 6. 课后教学
作业布置/提交/批改（自动+手动）、测评（定时测验）、学情报告（学生+班级）

### 7. 班级宠物积分
个人积分（多维度获取）、个人等级（6级）、班级宠物（全班喂养/成长进化）、徽章系统、防刷引擎

### 8. 多维度学情统计
编程能力/学习进度/创作能力/刷题数据/班级对比/AI 洞察

### 9. AI 能力（AI Gateway）
AI 作品点评、AI 创意题判题、AI 编程助手、AI 出题、AI 学情分析、AI 内容审核

### 10. 管理后台
用户管理、班级管理、内容审核（三级流水线）、数据统计
