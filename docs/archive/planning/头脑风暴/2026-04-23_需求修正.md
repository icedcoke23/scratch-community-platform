### 3. 教学功能面向课后
- ❌ 不做直播课、录播课
- ✅ 核心场景：
  - 作业布置/提交/批改
  - 测评（定时测验）
  - 班级宠物积分系统（趣味激励）
  - 学员多维度数据统计

### 4. OJ 核心能力：sb3 深度解析 ⭐
系统自动将 .sb3 文件解压，获取其中的 JSON 文件，然后解析提取：

| 提取内容 | 说明 |
|---|---|
| 程序链 | 积木序列、触发方式、嵌套结构 |
| 角色 | 名称、坐标、大小、方向、造型、声音 |
| 背景 | 造型列表、切换逻辑 |
| 声音 | 所有声音文件及时长 |
| 变量 | 全局/私有变量名称、类型、默认值 |
| 列表 | 全局/私有列表名称、默认值 |
| 广播 | 消息列表、发送/接收关系 |
| 素材文件 | 所有 png/svg/wav/mp3 及 md5 |

解析后的结构化数据发送给 AI/教师进行识别分析。

### 5. 只针对 Scratch
- 当前阶段只做 Scratch
- 后续扩展 Python、C++ 等
- 架构设计时预留语言扩展接口

## 修正后的技术要点

### sb3 解析是核心差异化

```
.sb3 = ZIP
  └── project.json (核心)
        ├── targets[]
        │   ├── [0] Stage (舞台)
        │   │   ├── variables: {id: [name, value]}
        │   │   ├── lists: {id: [name, []]}
        │   │   ├── broadcasts: {id: name}
        │   │   ├── blocks: {id: blockData}
        │   │   ├── costumes: [{name, md5ext, ...}]
        │   │   └── sounds: [{name, md5ext, ...}]
        │   └── [1..N] Sprites (角色)
        │       ├── name, x, y, size, direction, visible
        │       ├── variables, lists
        │       ├── blocks
        │       ├── costumes
        │       └── sounds
        ├── monitors[] (变量监视器)
        └── extensions[] (扩展)
```

解析后输出结构化 JSON，供 AI/教师/判题引擎使用。
