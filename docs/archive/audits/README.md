## 关键改进历程

### 安全加固
- JWT 密钥生产环境强制校验（拒绝默认密钥）
- BCrypt 密码加密 (strength=10)
- XSS 双重防护（Jackson XSS + XSS Filter）
- 幂等性拦截器防重复提交
- 敏感词 DFA 自动机
- API 路径遍历防护
- SB3 ZIP 路径遍历增强
- SSE 一次性 Token（防 JWT 泄露）
- @RateLimit 注解自定义限流

### 架构优化
- 模块化单体架构（10 个 Maven 模块）
- 事件驱动解耦（Spring Event + 降级机制）
- 沙箱进程隔离（Node.js 独立进程）
- 分布式锁（Redisson）+ 乐观锁（@Version）
- 滑动窗口限流（Redis Lua / 内存子窗口）
- Flyway 数据库版本管理（V1-V19）
- Caffeine 本地缓存高频查询

### 前端演进
- Vue 3 + TypeScript + Vite 全面建设
- Token 自动刷新（防并发竞态）
- 路由进度条 Vue 组件化
- ErrorBoundary 全局错误捕获
- PWA Service Worker
- 深色模式 + i18n
- 166 个单元测试

### 测试覆盖
- 后端：17 个测试类覆盖核心 Service
- 前端：14 个测试文件 / 166 个测试
- E2E：4 个 Playwright 测试
- 性能：4 个 k6 压测场景
